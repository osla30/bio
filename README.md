## 🚀 為什麼選擇科科計算工具箱？

在分秒必爭的實驗室環境中，我們追求的是 **Speed (速度)** 與 **Precision (精準)**：

* **⚡ 極速反應**：純網頁輕量化設計，手機或電腦瀏覽器打開即用，無需等待軟體載入。
* **🎯 實驗室專精**：針對生醫實驗常見的莫耳濃度換算、稀釋倍率、溶液配製、動物投藥、細胞計數等痛點深度優化。
* **📱 隨身助手**：響應式介面設計，讓您在 Bench 前拿著手機就能直接算出稀釋體積、投藥方式、細胞數量等。
* **🔒 隱私安全**：所有數據皆在本地端運算，實驗機密數據絕不上傳雲端，保障研究安全。

## 🛠️ 強大實驗計算功能概覽

工具箱內嵌多項專為生醫實驗情境設計的功能：


### 🎯 濃度、溶液配製計算
* **莫耳濃度換算**：輸入分子量與目標濃度、體積，自動產出所需藥品質量。
* **實驗稀釋計算**：直覺式填入數值，快速計算 Stock 稀釋比例。
* **酸鹼濃度計算**：快速計算酸鹼化學物質的重量百分比濃度與莫耳濃度。
* **多溶質配製計算**：可一次性填入多項溶質的分子量或原液濃度，快速產生詳細的溶液配製所需材料清單。


### 🎯 動物投藥、細胞數量計算
* **動物投藥計算**：輸入投藥方式、所需劑量，自動產出所需藥品配製方法。
* **細胞數量計算**：直覺式填入數值，自動計算出細胞數量，甚至可填入想要種的細胞數、份數，自動計算出所需數值，協助快速完成實驗。


### 🎯 分子生物學計算
* **Primer 回溶與稀釋**：配製 Primer 時，快速計算各種濃度與體積，加錯水體積時，也不會慌張。
* **Primer Tm值計算**：得知 Primer 大約的 Tm值，長度、GC%、二級結構與 Dimer。
* **DNA 反轉/互補**：輸入 DNA 序列，可進行反轉、互補、反轉+互補，同時具有去除 ATCG 以外字元的功能。
* **Ligation 計算**：快速計算 Insert 與 Vector 的莫耳比。
* **qPCR 相對定量分析**：輸入 qPCR Raw Data，快速以 2^(-ΔΔCt) 方法進行 qPCR 相對定量分析。


### 🎯 蛋白質相關計算
* **SDS-PAGE膠體配製計算**：支援 2X、4X Buffer 鑄膠，支援小編自製 TGOne 系統，同時也支援 BioRad TGX 系統。
* **Bradford/BCA 蛋白質定量**：Bradford/BCA 蛋白質定量計算、Western Blot 樣本配製計算。
* **Western Blot 條帶相對定量標準化**：輸入條帶灰度值，即可進行相對定量與標準化統計分析。
